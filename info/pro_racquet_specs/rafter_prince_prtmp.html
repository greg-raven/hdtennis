Pat Rafter’s Prince Precision Response Titanium MP

If memory serves, Pat Rafter for some time used the Prince Precision Response 660 MP. When Prince introduced the Precision Response Titanium MP, Rafter began playing with a burgundy-colored racquet that appeared to be the Precision Response Titanium MP. He later switched to a racquet that appeared to be a Prince Triple Threat Warrior MP (the white racquet).

I got my hands on one of Rafter’s Precision Response Titanium MPs, and it doesn’t seem to be any of those other racquets.

It’s the same length, beam thickness, and head size as the retail version of the Precision Response Titanium MP, but the other measurements are different from stock:
